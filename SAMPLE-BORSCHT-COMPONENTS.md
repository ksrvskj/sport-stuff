# Дизайн-система: компоненты и решения

## Актуальные компоненты (реализованы)

### Типографика
- **Заголовок H2** — название главы с номером в акцентном цвете
- **Подзаголовок H3** — uppercase, UI-шрифт, приглушённый
- **Абзац** — body font, 0.95rem, межстрочный 1.78
- **Ссылки** — акцентный цвет, тонкое подчёркивание (25%), на hover проявляется. Внешние → target="_blank"

### Навигация
- **Sidebar TOC (десктоп)** — sticky слева, подсветка текущей секции
- **Inline TOC (мобильный)** — компактный блок после хедера
- **Мобильная панель** — выезжает сверху при скролле вверх. Название главы → «СОДЕРЖАНИЕ» при открытии dropdown
- **Прогресс-бар** — тонкая полоска вверху (accent, 40% opacity). При мобильной панели переезжает на её нижний край
- **Стрелка вверх** — полупрозрачная (50% opacity), правый нижний угол

### Блоки контента
- **Preface** — тёплый фон + бордюр = вводный текст
- **Source-quote** — тонированный фон + акцентный левый бордюр + декоративная » = «чужие слова»
- **Explainer** — тёплый фон + акцентная полоска СВЕРХУ (3px) + лейбл «Контекст» = «редакция объясняет»
- **Blockquote** — серый фон + серый левый бордюр = «справочные данные»
- **Key-points** — нейтральный бокс с тире-списком и жирными выводами
- **Why-matters** — итоговый блок внизу главы, отделён верхней линией
- **Draft-note** — жёлтый пунктир, «✎ заметка редактора». Убирается перед публикацией
- **Collapsible** — нативный <details>, стрелка ▸/▾, для длинных цитат/документов

### Списки и данные
- **Маркированный список** — тире (—) в акцентном цвете
- **Definition list** — термин → стрелка → значения, один шрифт
- **Timeline** — вертикальная линия, точки, дата + событие

### Медиа
- **Изображение с подписью** — figure + figcaption, рамка, подпись курсивом

### Интерактив
- **Сноски** — hover на десктопе, клик на мобильном. Тёплый tooltip, auto-flip (вверх/вниз), auto-anchor (лево/право). Fallback: список внизу главы

### Мета
- **Блок авторов** — жирная линия, список имён + анонимные курсивом
- **Настройки** — footer с 6 шрифтовых пар и 4 цветовых палитры

### Адаптивность
- **Десктоп (>960px)** — sidebar + контент
- **Планшет (640–960px)** — без sidebar, шире, мобильная панель
- **Телефон (<640px)** — компактнее, мельче шрифт

### Safari iOS
- viewport-fit=cover, theme-color, safe-area-inset-bottom

## Можно попробовать позже

### Lead paragraph
Первый абзац главы чуть крупнее (1.05rem). Риск: выглядит как баг.

### Внутренние ссылки между главами
«см. главу 4» — пока обычные якоря. Можно добавить § перед номером.

### Embed-превью YouTube/Telegram
YouTube: thumbnail + play + ссылка (без iframe). Telegram: мини-карточка вручную.

### Drop cap (буквица)
Для аналитики — претенциозно. Пока нет.

### Прогресс по главам
«Глава 4 из 8» в мобильной панели. Возможно overkill.

### Тёмная тема
CSS variables готовы, нужно продумать все блоки.

### Кнопка «поделиться»
Share API (мобильный) / копировать ссылку (десктоп). Можно с якорем на главу.

### Печать
Базовый @media print есть. Доработка: убрать draft-notes, раскрыть collapsible, сноски как footnotes.

### Инлайн-цитаты
Решение: не стилизовать, обычные кавычки. Различие обеспечивают блочные цитаты.
